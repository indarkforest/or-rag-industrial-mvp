# 苯乙烯悬浮聚合工艺规程（节选）

## 1. 工艺参数控制指标

| 参数 | 测点 | 控制指标 | 说明 |
|---|---|---|---|
| 聚合温度 | TI-101 | 85±3℃ | 温度过高导致爆聚风险 |
| 釜内压力 | PI-101 | 0.25~0.35 MPa | 压力随温度升高而上升 |
| 冷却水流量 | FI-102 | 18~22 m³/h | 保证移热能力 |
| 搅拌转速 | SI-101 | 90~110 rpm | 保证悬浮分散与传热 |
| 引发剂加入量 | - | 0.35% (wt) | 过量将加剧放热 |

## 2. 温度控制要求

聚合反应为强放热反应，反应热约 670 kJ/kg。温度控制回路 TIC-101 通过调节冷却水阀 TV-101 开度维持釜温。
反应中期（转化率 30%~60%）放热速率最大，此阶段必须保证冷却水流量不低于 18 m³/h。

## 3. 工艺异常处置

- 釜温超过 90℃ 时：应立即检查 TV-101 阀位反馈与 FI-102 流量，手动开大冷却水阀。
- 引发剂误加过量时：放热速率超出冷却能力，即使冷却水全开也可能出现温度飞升，必须启动紧急加料终止剂程序。
- 釜温超 98℃：联锁系统自动动作，停止单体进料并全开紧急冷却水阀 XV-105。

## 4. 质量影响

聚合温度偏离 85±3℃ 控制带会导致分子量分布变宽，产品熔指不合格。
温度长期偏高还会造成粘釜，增加清釜频次。
